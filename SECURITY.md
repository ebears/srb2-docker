# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| latest  | :white_check_mark: |

Only the latest Docker image is supported. Users should always pull the most recent image to receive security fixes.

## Reporting a Vulnerability

Report security vulnerabilities through [GitHub Security Advisories](https://github.com/ebears/srb2-docker/security/advisories/new).

Please do **not** open public issues for security vulnerabilities.

We aim to acknowledge reports within 3 business days and provide a fix or mitigation within 30 days of triage.
